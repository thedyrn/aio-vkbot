# Aio-vkbot
Async bot framework for vk.com, with asyncio and aiohttp.

### В разработке
* Автомат конечных состояний для диалогов
* Поддержка CallbackAPI
* Поддержка всех возможностей сообщений

### Тестируется
* LongPolling
* Обработчики обновлений

### Стек
* Python 3.8
* asyncio
* aiohttp
* pytest
